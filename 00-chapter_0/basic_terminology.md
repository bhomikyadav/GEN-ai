# 2. Machine Learning (ML)

Instead of programming every rule manually, we train a model using data.

Traditional Programming

```
Input + Rules
      ↓
 Output
```

Machine Learning

```
Input + Output
      ↓
 Training
      ↓
Model learns rules
```

Example

Spam Detection

Instead of

```
if(email contains "lottery")
```

We train using

```
100,000 emails
Spam
Not Spam
```

# 5. Large Language Model (LLM)

An LLM is a huge neural network trained on massive amounts of text.

It predicts the next word repeatedly.

Example

Input

```

```

Possible prediction

```
pizza
```

Then

```
I love eating pizza
```

Then

```
I love eating pizza because...
```

Examples

* OpenAI GPT
* Anthropic Claude
* Google Gemini
* Meta Llama

# 10. Token

This is one of the most important concepts.

LLMs don't read words.

They read **tokens**.

Example

```
Hello World
```

may become

```
Hello
World
```

or even smaller pieces depending on the tokenizer.

Example

```
Programming
```

may become

```
Program
ming
```

Every API call is billed and limited by **tokens**, not characters or words.

---

# 11. Tokenization

Process of converting text into tokens.

```
Sentence

↓

Tokenizer

↓

Tokens
```

# 12. Context Window

The amount of text an LLM can remember during one conversation.

Example

```
10 pages
```

or

```
100 pages
```

depending on the model.

If the conversation exceeds the context window, older information may be dropped.

# 18. Embeddings

Embeddings convert text into numerical vectors that capture semantic meaning.

Example

```
Cat

↓

[0.23, -0.81, 0.55, ...]
```

Texts with similar meanings have vectors that are close together.

Used for

* Semantic Search
* Recommendations
* Retrieval-Augmented Generation (RAG)

---

# 19. Vector

A vector is simply an array of numbers representing meaning.

Example

```
[0.45, -0.91, 0.32, ...]
```

---

# 22. RAG (Retrieval-Augmented Generation)

Instead of relying only on the model's knowledge:

```
Question

Vector Search
      ↓
Relevant Documents
      ↓
LLM
      ↓
Answer
```

This helps the model answer using your own documents.
